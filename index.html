How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Note: This is the special version of 2048 game to help people feel the rules of color mixing. Adapter is Wazons from SJTU-Biox-Shanghai.

Original 2048 game is created by Gabriele Cirulli. Based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.
